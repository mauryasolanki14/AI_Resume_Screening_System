# 🚀 AI Resume Screening System

An intelligent AI-powered system that analyzes resumes against job descriptions using NLP, Machine Learning, and LLMs.

---

## 🔥 Features

- 📄 Upload Resume (PDF)
- 🧠 AI-based Resume Analysis
- 📊 Match Score Calculation
- ✅ Matched Skills Extraction
- ❌ Missing Skills Identification
- 💡 Smart Suggestions for Improvement
- ⚡ Hybrid Model (Embeddings + LLM)

---

## 🧠 Tech Stack

- Python
- Flask
- LangChain
- Ollama (LLaMA3)
- Sentence Transformers
- HTML, CSS

---

## ⚙️ How It Works

1. Resume is parsed and cleaned
2. Job description is processed
3. Embedding model calculates similarity
4. LLM analyzes context and extracts insights
5. Hybrid scoring system generates final result
